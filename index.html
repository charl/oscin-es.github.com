Oscines is a JavaScript library based on the standard combinators as well as a number of utility combinators and decorators introduced in the book JavaScript Allongé.

The Code

There are three kinds of functions in the library. Functions with the capitalized nickname of a bird are fixed arity forms of the combinators, e.g. `function Cardinal (a, b, c)`, `function Thrush (a, b)`.

Functions with a capitalized abbreviated name are curried forms that match the semantics of combinatory logic, e.g. `C` is the curried form of `Cardinal`. Functions with lower-case names are practical JavaScript combinators and function decorators, e.g. `once`, `variadic`.

Songbirds

To Mock a Mockingbird established the metaphor of songbirds for the combinators, and ever since then logicians have called the K combinator a "kestrel," the B combinator a "bluebird," and so forth.< >/p>

In Raymond Smullyan's book, when you call out the song of a bird, each songbird responds with a song denoting a songbird. For example, the Identity bird sings back the name of the bird you sang to it. From the presence of a few simple birds in a forest, you can deduce all of programming.

Project Status

Under construction, barely an alpha. All of the combinators and function decorators from JavaScript Allongé are present. The logical combinators from Bluebird to Owl are present. There are no tests or further documentation.
