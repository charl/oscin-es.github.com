Oscines is a JavaScript library based on the standard logical combinators as well as a number of practical combinators and function decorators based on the book JavaScript Allongé.

oscines.js.

Project Status

Under construction, barely an alpha.

Songbirds

Raymond Smullyan's book To Mock a Mockingbird established the metaphor of songbirds for the combinators, and ever since then logicians have called the K combinator a "kestrel," the B combinator a "bluebird," and so forth.
